# Stimulus Source

The reading stimulus was an excerpt from the opening of Ryunosuke Akutagawa's short story "Hana" ("The Nose").

## Source Metadata

- Work: "Hana" ("The Nose")
- Author: Ryunosuke Akutagawa
- Source: Aozora Bunko
- URL: https://www.aozora.gr.jp/cards/000879/files/42_15228.html
- Base text: `Akutagawa Ryunosuke Zenshu 1`, Chikuma Bunko, Chikuma Shobo
- First printing of base text: September 24, 1986
- Fourteenth printing used by Aozora Bunko: April 15, 1997
- Parent base text: `Chikuma Zenshu Ruiju-ban Akutagawa Ryunosuke Zenshu`, Chikuma Shobo, March-November 1971
- First publication: `Shinshicho`, February 1916

## Excerpt Used

The experiment used the opening excerpt of "Hana" beginning with the description of Zenchi Naigu's nose and ending with the explanation that Naigu suffered because of the self-esteem injured by his nose.

The exact Japanese excerpt is maintained in the private study record. It is not duplicated in this English repository to keep reviewer-facing repository documentation in English.
